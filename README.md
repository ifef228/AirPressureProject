Разработка интернет-приложений (РИП) 2025
Расчет температуры атмосферы
Разработанное приложение позволит синоптикам создавать и формировать заявки на расчёты температуры атмосферы, а менеджерам по работе с заявками завершать или отклонять данные заявки.
